Privacy Policy — Newbook Toolkit 2.0

Last updated: 6 May 2026

Overview

Newbook Toolkit 2.0 is a browser extension built by the Newbook Implementation Team (Storable) for internal use by Newbook staff. It provides workflow automation tools for client onboarding and property management system configuration.

Data Collection

Newbook Toolkit 2.0 does not collect, transmit, or sell any personal data to third parties. All user data remains within systems already authorised by the user's organisation.

What the extension stores locally

The following data is stored in your browser's local storage (chrome.storage.local) and never leaves your device:

  • Google OAuth token (for authenticated API access)
  • Atlassian email and API token (for Confluence integration)
  • Monday.com API token (for bug/feature reporting)
  • Extension preferences and UI state
  • Labour tracking session data (timers, client IDs)
  • Wellness tracker settings (break intervals, reminders)
  • Meeting reminder preferences

What the extension communicates with

The extension communicates only with the following services, using credentials you explicitly provide:

Authentication

  • Google: Uses Chrome's built-in chrome.identity OAuth 2.0 flow. The extension requests only the scopes necessary for its features. You can revoke access at any time via your Google Account permissions.
  • Atlassian: Uses HTTP Basic Auth with an API token you generate and provide. Stored locally in your browser only.
  • Monday.com: Uses a personal API token you provide. Stored locally in your browser only.

Permissions

The extension requests only the permissions necessary for its functionality. See the Chrome Web Store listing for a full breakdown of each permission and its justification.

Data Retention

All locally stored data can be cleared at any time by:

  • Disconnecting accounts within the extension's "Configure Connections" panel
  • Removing the extension from your browser
  • Clearing browser extension storage

No data persists on any external server controlled by this extension.

Third-Party Services

This extension does not use analytics, tracking pixels, advertising networks, or any third-party data collection services.

Who Can Use This Extension

This extension is intended for use by authorised Newbook (Storable) staff only. It requires valid credentials for Newbook Cloud and related internal systems to function.
